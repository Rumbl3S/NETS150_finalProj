Short report — graph model, algorithms, example outputs
=========================================================

Graph model
-----------
Nodes are movies (integer ids aligned with the MovieLens-style `movies.csv`). Undirected weighted edges use weight (1 − similarity), so more similar neighbors yield shorter Dijkstra paths.

**Toy / enriched CSVs:** Similarity can blend genre Jaccard, actor Jaccard and director match (from optional `movie_people.csv`), plus Pearson correlation on users who rated both films (minimum overlap threshold).

**MovieLens 20M at full scale:** Cast and director are not in the core GroupLens files, so we use **genre Jaccard plus rating correlation** only (fixed blend). Ratings are **streamed** from `ratings.csv`; each movie keeps a **bounded random sample** of (user, rating) pairs for correlation so the full ~20M rows never load into memory. The graph is **sparse**: each movie considers genre neighbors, refines with a two-stage score, and keeps only the strongest **top edges per movie**, merging parallel suggestions with the **minimum** edge weight so Dijkstra sees the best link between two movies.

Algorithms
----------
• Breadth-first search (BFS) and depth-first search (DFS) traverse the adjacency list in deterministic ascending neighbor id order; they ignore edge weights and illustrate classic exploration from a start movie (cap raised to 50 nodes in the UI for large graphs).
• Recommendations use multi-source Dijkstra: every selected “liked” movie is a source with initial distance zero; the algorithm relaxes edges in best-first order until shortest-path distances are known. Movies are ranked by increasing distance from the seed set (excluding the seeds themselves).

Example outputs
---------------
Seeding with “Toy Story (1995)”, “Jumanji (1995)”, and “Grumpier Old Men (1995)” produces top recommendations including “Sabrina (1995)” (distance 0.525), “Tom and Huck (1995)” (0.610), and “American President, The (1995)” (0.642) — all 1995 family/comedy-adjacent titles that share genre overlap and rating correlation with the seed set.

Running BFS from “Toy Story (1995)” visits Toy Story first, then expands through adjacent neighbors: Jumanji, Tom and Huck, GoldenEye, Dracula: Dead and Loving It, Balto, Powder — reflecting the ring-by-ring structure of the similarity graph.

Running DFS from “Toy Story (1995)” follows one branch of the graph in preorder — Jumanji, Tom and Huck, GoldenEye, Heat, Sudden Death — before backtracking, producing a noticeably different visit order than BFS despite the same start node.
