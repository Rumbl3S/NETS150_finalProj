Short report - graph model, algorithms, and example outputs
===========================================================

Graph model
-----------
Nodes are movies, identified by integer ids that match the MovieLens movies.csv
file. The graph is undirected and weighted, with weight = 1 - similarity, so
more similar movies have shorter Dijkstra distances.

For the bundled toy CSVs, similarity blends three signals: genre Jaccard, actor
Jaccard and director match (from the optional movie_people.csv), and Pearson
correlation on users who rated both films (subject to a minimum overlap
threshold).

For the full MovieLens 20M release, cast and director are not in the GroupLens
core files, so similarity uses genre Jaccard plus rating correlation only.
Ratings are streamed from ratings.csv rather than loaded into memory, and each
movie keeps a bounded reservoir sample of (user, rating) pairs for the
correlation computation. The graph is sparse: each movie keeps only its top-N
strongest neighbors, and parallel edges between the same pair of movies are
merged using the minimum weight so Dijkstra always sees the best available link.

Algorithms
----------
- BFS and DFS traverse the adjacency list in deterministic ascending neighbor-id
  order. They ignore edge weights and illustrate classic graph exploration from
  a chosen start movie. The UI caps the printed traversal at 50 nodes for
  readability on the larger graph.

- Recommendations use multi-source Dijkstra. Every movie in "Your picks" is a
  source with initial distance zero, and the algorithm relaxes edges in
  best-first order until shortest-path distances from the seed set are known.
  Movies are then ranked by increasing distance, with the seeds themselves
  excluded from the output.

Example outputs (from the included screenshots)
-----------------------------------------------
Seeding with "Toy Story (1995)", "Jumanji (1995)", and "Grumpier Old Men (1995)"
produced top recommendations of "Sabrina (1995)" (distance 0.525), "Tom and
Huck (1995)" (0.610), and "American President, The (1995)" (0.642). All three
are 1995 family / comedy-adjacent titles that share genre overlap and rating
correlation with the seed set, which matches the intuition that the graph is
clustering similar movies close to one another.

Running BFS from "Toy Story (1995)" visited Toy Story first and then expanded
through its immediate neighbors: Jumanji, Tom and Huck, GoldenEye, Dracula:
Dead and Loving It, Balto, and Powder. The output reflects the ring-by-ring
behavior of BFS on the similarity graph.

Running DFS from the same start node followed one branch in preorder
(Jumanji, Tom and Huck, GoldenEye, Heat, Sudden Death) before backtracking,
producing a noticeably different visit order than BFS even though the start
node was identical.
